# PhD.Hub Landing Page

A modern, responsive landing page for **PhD.Hub** - a Vietnamese academic community platform founded by Dr. Tran Quoc Thien. This website connects over 110,000+ members worldwide to share scholarships, academic experiences, and support Vietnamese students on their journey to international education.

PhD.Hub is an academic community that provides:
- **Scholarship Information**: Centralized platform for searching and sharing full scholarship opportunities at all academic levels
- **Community Support**: Network of Vietnamese professors and researchers from developed countries providing mentorship and guidance
- **PhD.Hub Foundation**: A volunteer fund supporting disadvantaged Vietnamese youth through scholarships, academic resources, and direct mentorship
- **MacroMentorship**: Long-term career guidance program focusing on developing critical thinking, independent work, and strategic planning skills

## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required - pure HTML, CSS, and JavaScript

### Installation & Usage

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd landing_page
   ```

2. **Open the website**
   - Simply open `index.html` in your web browser
   - Or use a local development server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Or using Node.js
     npx serve
     ```

3. **Navigate**
   - Visit `http://localhost:8000` in your browser
   - Explore different pages through the navigation menu
